## Uppgiftens Syfte

- Att bygga en applikation med **TypeScript**
- **Objektorienterad programmering (OOP)** med klasser och konstruktorer
- Användning av **interface** för att definiera datastrukturer
- **LocalStorage** för att spara och hämta data persistent

## Uppgiftens lösning
-- Användare kan lägga till nya task via formulär på sidan **Lägg till todo** 
-- Validering gör att användare lämnar ingen tomt fält eller att task är sparad 
-- Alla task sparas i en tabell på startsidan **Todolista** med kolumner för uppgift, prioritet, status och radering. 
-- Varje uppgift kan markeras som klar genom att klicka på knappen **Slutför** vilket ändrar status till **Klar** 
-- Task kan raderar genom att klicka på knappen **Radera** 
-- Alla task sparas i **LocalStorage** 
-- **Responsive design** för olika skärmstorlekar
